docker常用命令

2019-11-14

  • 拉镜像

    docker pull registry.cn-hangzhou.aliyuncs.com/helowin/oracle_11g

  • 创建容器信息

    docker run -d -p 1521:1521 --name oracle_11g registry.aliyuncs.com/helowin/oracle_11g

  • 进入容器shell

    docker exec -it oracle_11g bash

  • 运行

    docker start oracle_11g
