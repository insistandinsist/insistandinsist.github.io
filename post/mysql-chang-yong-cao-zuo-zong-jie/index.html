mysql常用操作总结

2019-11-28

  • 改密码

    在mysql 8前有效

      set password for 'root'@'localhost'=password('新密码');

    mysql 8及以后的代码

      use mysql;
  alter user 'root'@'localhost' identified with mysql_native_password by '新密码';
  flush privileges;

    原因是：

    在之前，mysql的密码认证插件是"mysql_native_password",现在使用的是"caching_sha2_password"

  • 导入sql文件

    若使用navicat导入文件失败，可以使用以下代码

      mysql -uroot -p 数据库名 < xxx.sql
